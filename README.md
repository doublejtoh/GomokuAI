# GomokuAI
Gomoku AI: RuleBased vs DQN(2015)

## Results

over 5000 episodes, DQN won about 440 episodes against RuleBased AI. <br>
When DQN won, It took long steps(mostly about 220 steps) otherwise took short steps(mostly 20 steps).
